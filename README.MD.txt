# Tichi QA Assignment

## Overview

This repository contains a QA assignment for the Tichi application. The assignment focuses on manual testing by creating test cases and defect reports for the Login and Signup functionalities.

## Tasks Completed

### Task 1: Test Case Creation

* Login Test Cases
* Signup Test Cases
* Positive and Negative Test Scenarios
* Validation and Security Test Cases

### Task 2: Defect Reporting

* Created professional defect reports
* Included severity, priority, expected result, and actual result
